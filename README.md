# grafana-loki-alloy-docker-demo

![Grafana Alloy Loki on Docker for Homelab Monitoring With A Farmer](https://raw.githubusercontent.com/MarcusHoltz/marcusholtz.github.io/main/assets/img/header/header--grafana--alloy-loki-stack-docker-homelab-monitoring.jpg)

Alloy, Loki, and Grafana up and running with a quick guide for monitoring Docker environments. Supports Docker container logs, Traefik JSON access logs, GeoIP enrichment, hybrid local/cloud deployment, with automated provisioning. Includes complete configuration examples and setup script.
